# Roadmap Governance

Phase 7 adds a JSON-backed roadmap registry for core, analysis, tactical, reliability, operations, source intake, source knowledge, and governance features.

Roadmap items track status, risk, dependencies, tests, docs, API, UI, storage, and warnings.

Statuses:

```text
planned
in_progress
implemented
verified
blocked
deferred
deprecated
unknown
```

Unknown features are reported as `unknown`; they are not silently ignored.
